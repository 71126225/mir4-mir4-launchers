Command / Create and organize

1. Moonju exclusive command
  • Alliance: Command for clans. The clans that want to form an alliance must meet, stand right next to each other, face each other, and use the following command.
    • Example) @Alliance Mir's Legend
  • To be hostile: Command from the sect leader. If there is a sect you wish to be hostile to, do the following.
    • Example) @War Mir's Legend
  • To disband: Command for clans. To disband, clansmen must meet, stand right next to each other, face each other, and use the following command.
    • Example) @Cancel War Legend of Mir
  • Clan war cry: Clan member command. Heard by all members on other servers.
    • Example) /> Hello, this is Legend of Mir.
  • @Clan Notice: A command to enter notices that are visible to people in the same clan when logging in.
  • @Title Name Title Name: A command that gives a title to the person whose name is given.
    Note) Only those who are given the title and are level 70 or higher and ranked 5th or higher within the sect can use it.
  • @Cult position name sequence [0...10] Position name: Command to change the position of a cult member
               Example) @Cult Position Name Rank 0 Cult Leader
  • @Move the sect's order [0...10] Sect member name: Command to change the sect member order
               Example) @Moonpa Seol-i-dong Seol-i-seol 0 Geum-seok

2. Moonwon full command
  • @Cult information: Command to check the entire cult
  • @Sect: Command to check your sect
  • @Sect withdrawal Character name: Command used when you want to withdraw from a sect
  • @> : Speaking to the entire sect


3. Division stock and above commands
  • @Cult Join Character Name: Command to join the cult with the given name
